# Session Replay for React Native

Session Replay support for the TrueWatch React Native SDK.

Use this package together with `@truewatchtech/react-native-mobile` to capture user sessions and protect sensitive UI content with configurable privacy rules.

## Installation

```sh
npm install @truewatchtech/react-native-session-replay
```

## Usage

```js
import {
  import {
  FTReactNativeSessionReplay,
  FTSessionReplayConfig,
  SessionReplayPrivacy,
} from '@truewatchtech/react-native-session-replay';

// ...

```
